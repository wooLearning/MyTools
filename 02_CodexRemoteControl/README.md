# Codex Remote Control

Windows Codex Desktop을 모바일에서 원격 제어할 수 있게 켜는 도구입니다.

## 사용법

1. Codex Desktop을 한 번 실행하고 로그인합니다.
2. `CodexRemoteControl.exe`를 실행합니다.
3. `.codex folder`는 보통 기본값 그대로 둡니다.
4. `Activate`를 누릅니다.
5. Codex Desktop을 껐다가 다시 켭니다.
6. `상태 확인`을 눌러 `Selected DB`가 `ON`인지 확인합니다.

## 주의

- 이 도구는 원격 기능을 켜기만 합니다. 끄기 기능은 없습니다.
- PC의 Codex Desktop과 모바일 ChatGPT/Codex 앱은 같은 계정이어야 합니다.
- 서명되지 않은 exe라 Windows 경고가 뜰 수 있습니다.
